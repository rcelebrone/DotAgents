Skill: Onboarding & Scaffolding

1. Descrição

Gera boilerplate de código seguindo o padrão de design do projeto atual.

2. Templates Disponíveis

C#: Entity, Repository, Unit Test (xUnit), ValueObject.

Dart: Entity, DataModel, State Management Class, Widget Component.

JS: Service, Route, Middleware, Integration Test.

3. Workflow

Ao receber "Crie uma entidade [Nome]", o agente deve gerar todos os arquivos relacionados para que a feature já comece funcional e testável.
