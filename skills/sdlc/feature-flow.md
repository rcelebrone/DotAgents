# Skill: Feature Flow & Scaffolding

## Workflow
1. **Branching**: Cria branch `<type>/<kebab-case>` (Tipos: feat, fix, refactor, chore).
2. **Tasking**: Gera arquivos `001-nome.md` em `.agents/docs/tasks/`.
3. **Scaffolding**: Gera boilerplate (Entities, Controllers, Services) seguindo o padrão injetado no `architecture.md`.
